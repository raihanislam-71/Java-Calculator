# 🧮 Java Calculator

A simple desktop calculator built with **Java Swing**. This project performs basic arithmetic operations with a clean and user-friendly interface.

## 📸 Preview

### Calculator UI

![Calculator UI](https://github.com/raihanislam-71/Java-Calculator/blob/d435ae0298a07f45d1671620342321515826c1cb/Screenshot%202026-07-18%20154012.png)

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- √ Square Root
- +/- Positive / Negative Toggle
- ⌫ Backspace
- 🗑 Clear Display
- 🔢 Decimal Number Support
- ⚠ Divide by Zero Error Handling

## 🛠 Technologies Used

- Java
- Java Swing
- AWT
